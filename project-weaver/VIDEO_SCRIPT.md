## RECORDING TIPS

### Visual Guidelines
1. **Keep the dashboard open** - Show the UI throughout, not just slides
2. **Use smooth transitions** - Fade between tabs/views
3. **Highlight cursor movement** - Make it easy to follow along
4. **Show real data** - Use the demo mode we just built
5. **Add subtle animations** - When showing features appearing

### Audio Guidelines
1. **Speak clearly and confidently** - You're explaining something revolutionary
2. **Vary your pace** - Slow down for key points, speed up for transitions
3. **Use music** - Upbeat, tech-focused background track (low volume)
4. **Add sound effects** - Subtle whooshes for transitions

### Technical Setup
1. **Resolution:** 1920x1080 minimum
2. **Frame rate:** 30fps or 60fps
3. **Screen recording tool:** OBS Studio, QuickTime, or Loom
4. **Audio:** External mic recommended (Blue Yeti, Rode, etc.)
5. **Lighting:** Good lighting if showing yourself
