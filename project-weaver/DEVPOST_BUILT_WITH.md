# Built With

## AI & LLM
- **Google Gemini 2.0 Flash** - Core AI reasoning engine powering all 5 specialized agents
- **Model Context Protocol (MCP)** - Standard protocol for AI agent tool integration

## Frontend
- **Next.js 15** - React framework with App Router and Server-Side Events (SSE)
- **React 19** - Latest React with concurrent features
- **TypeScript** - Type-safe development across the entire stack
- **Tailwind CSS 4** - Utility-first styling with modern design system
- **React Icons** - Heroicons for consistent UI

## Backend & Tools
- **Node.js** - Runtime for MCP server and Hub server
- **Express.js** - Hub server for team synchronization
- **Tree-sitter** - Multi-language AST parsing for code intelligence
- **@modelcontextprotocol/sdk** - Official MCP SDK for tool registration

## Code Intelligence
- **tree-sitter-typescript** - TypeScript/JavaScript AST parsing
- **tree-sitter-python** - Python code analysis
- **tree-sitter-java** - Java code parsing
- **tree-sitter-go** - Go language support
- **tree-sitter-rust** - Rust code analysis

## Architecture & DevOps
- **Git** - Version control and team collaboration backbone
- **JSON-based persistence** - `.weaver/` directory for shared state
- **Server-Sent Events (SSE)** - Real-time dashboard updates
- **RESTful API** - Next.js API routes for data access

## Key Technical Innovations
- **Persistent Agent Memory** - Shared context board across all agents
- **AST-Powered Indexing** - Deep code understanding without re-parsing
- **Human-in-the-Loop Verification** - Trust building through annotation validation
- **Git-Native Collaboration** - Zero-config team sharing via version control
- **55 MCP Tools** - Comprehensive toolkit spanning 19 modules
